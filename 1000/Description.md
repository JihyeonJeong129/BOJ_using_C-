## 문제 출처

[BAEKJOON Online Judge #1000](https://www.acmicpc.net/problem/1000)

<br>


## 문제


두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.


<br>


## 입력


첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)


<br>


## 출력


첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)


<br>


## 예제


입력

```
1 2

```

<br>

출력

```
3

```

<br>
