## 문제

<br>


## 입력


첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)


<br>


## 출력


첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)


<br>


## 예제


입력

```python
3 2

```

<br>

출력

```python
1
```
